Float 속성 사용

div 1
div 2-Float 속성을 사용하여 대상 요소를 웹 문서에 배치하면 그 다음 요소에도 똑같은 속성이 적용된다. 하지만 float 속성이 사용되는 것을 원하지 않을 때도 있다. 이 때 clear 속성을 사용한다. 다양한 레이아웃을 설계할 때는 float 속성과 clear 속성을 적절히 잘 사용해야 한다.

Clear 속성 사용

div 3
div 4-Clear 속성은 float 속성이 적용되는 것을 원하지 않는 요소에 사용하여 float 속성을 초기화시킨다. float:left;를 사용했다면 clear:left;로 float:right;를 사용했다면 clear:right;로 초기화한다. Float 속성 값이 left인지 right인지 상관없이 무조건 초기화하고싶다면 clear:both;를 사용한다. 보통 clear:both;를 많이 사용한다.
